# Yandex-AI-Agents
Three-agent shopping assistant built with LangChain and OpenAI tool calling. Includes: (1) ReAct agent with product search and cart tools, (2) memory agent with JSON-persistent user profile, (3) multi-agent pipeline — Coordinator, Retriever, Pros/Cons, Ranker, Cart agents.

Implementation of AI agent patterns using LangChain: tool-calling ReAct loop, long-term memory via JSON persistence, and a multi-agent system for product recommendation. Built on an electronics catalog (headphones, keyboards, mice).
